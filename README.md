# 📊 Customer Churn Analysis using Python

## 📖 Overview

This project performs Exploratory Data Analysis (EDA) on the Telco Customer Churn dataset using Python. The goal is to understand customer behavior, identify churn patterns, and generate business insights through data visualization.

---

## 🛠️ Tools & Libraries

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## 📂 Dataset

- Telco Customer Churn Dataset

---

## 📊 Project Workflow

1. Data Loading
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Business Insights

---

## 📈 Visualizations

- Customer Churn Distribution
- Customer Distribution by Contract Type
- Customer Distribution by Payment Method
- Customer Distribution by Internet Service
- Distribution of Monthly Charges
- Senior Citizen Distribution
- Customer Gender Distribution
- Customer Tenure Distribution

---

## 💡 Key Business Insights

- Most customers remained with the company, while a smaller percentage churned.
- Month-to-month contracts are the most common among customers.
- Electronic check is the most frequently used payment method.
- Fiber optic and DSL are the primary internet services.
- Monthly charges vary across customers.
- The customer base is balanced by gender.
- Most customers are non-senior citizens.
- The dataset includes both new and long-term customers.

---

## 🎯 Conclusion

This project demonstrates how Python can be used for data cleaning, exploratory data analysis, and visualization to generate meaningful business insights. The findings can help organizations better understand customer behavior and support data-driven decision-making.
